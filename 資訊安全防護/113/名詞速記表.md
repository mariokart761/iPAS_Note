# 容易混淆名詞對照速記表

## 1. 弱點 / 威脅 / 風險 / 事件

| 名詞               | 速記             | 容易混淆點            |
| ---------------- | -------------- | ---------------- |
| 弱點 Vulnerability | 系統本身有洞         | 是「可被利用的缺陷」       |
| 威脅 Threat        | 會來利用洞的東西       | 可以是人、攻擊行為、惡意程式   |
| 風險 Risk          | 弱點被威脅利用後的影響可能性 | 不是單指漏洞本身         |
| 事件 Incident      | 已經發生的異常或損害     | 不是「可能發生」，而是「已發生」 |

**速記：**
洞 = 弱點；壞人/攻擊 = 威脅；出事機率與後果 = 風險；真的出事 = 事件

---

## 2. 弱點掃描 / 滲透測試 / 源碼掃描 / 資安健診

| 名詞   | 速記           | 容易混淆點       |
| ---- | ------------ | ----------- |
| 弱點掃描 | 找已知漏洞        | 偏自動化        |
| 滲透測試 | 模擬攻擊驗證能不能打進去 | 不是單純掃描      |
| 源碼掃描 | 看程式碼本身有沒有缺陷  | 不一定會真的打系統   |
| 資安健診 | 整體盤點與檢查      | 範圍通常比單一掃描更廣 |

**速記：**
掃描 = 找洞；滲測 = 打洞；源碼 = 看程式；健診 = 全身檢查

---

## 3. 網路弱點掃描 / 主機弱點掃描 / 網站弱點掃描

| 類型     | 速記           | 常考內容             |
| ------ | ------------ | ---------------- |
| 網路弱點掃描 | 看網路設備與服務暴露面  | 開放埠、服務版本、通訊協定    |
| 主機弱點掃描 | 看單一主機設定與修補狀況 | 修補、服務、帳號、系統設定    |
| 網站弱點掃描 | 看 Web 應用程式漏洞 | SQLi、XSS、上傳、敏感目錄 |

**速記：**
網路掃「設備與服務」；主機掃「作業系統與設定」；網站掃「Web 漏洞」

---

## 4. CVE / CVSS

| 名詞   | 速記      | 容易混淆點        |
| ---- | ------- | ------------ |
| CVE  | 漏洞編號    | 是漏洞 ID，不是分數  |
| CVSS | 漏洞嚴重度評分 | 是風險分數，不是漏洞名稱 |

**速記：**
CVE = 編號；CVSS = 分數

---

## 5. SAST / DAST

| 名詞   | 速記       | 容易混淆點      |
| ---- | -------- | ---------- |
| SAST | 靜態原始碼分析  | 不需實際執行程式   |
| DAST | 動態應用程式測試 | 要在執行中的系統上測 |

**速記：**
SAST 看原始碼；DAST 打執行中的程式

---

## 6. Brute Force / Password Spray

| 名詞             | 速記           | 容易混淆點   |
| -------------- | ------------ | ------- |
| Brute Force    | 對一個帳號試很多密碼   | 集中打一個帳號 |
| Password Spray | 用少數常見密碼試很多帳號 | 分散打一堆帳號 |

**速記：**
暴力破解 = 一人很多密碼；密碼噴灑 = 一個密碼噴很多人

---

## 7. Phishing / MFA / Phishing-resistant MFA

| 名詞                     | 速記      | 容易混淆點                 |
| ---------------------- | ------- | --------------------- |
| Phishing               | 釣魚騙帳密   | 不只 email，也可能簡訊、網站、即時通 |
| MFA                    | 多因子驗證   | 不代表一定抗釣魚              |
| Phishing-resistant MFA | 抗釣魚 MFA | 如 FIDO、PKI-based MFA  |

**速記：**
有 MFA 不一定安全到防釣魚；抗釣魚 MFA 才是進階版

---

## 8. SSO / MFA / VPN

| 名詞  | 速記       | 容易混淆點           |
| --- | -------- | --------------- |
| SSO | 單一登入     | 解決登入便利，不等於強身分驗證 |
| MFA | 多因子驗證    | 解決身分確認強度        |
| VPN | 安全遠端連線通道 | 解決通訊路徑，不等於帳號就安全 |

**速記：**
SSO 管方便；MFA 管身分；VPN 管通道

---

## 9. MITM / DDoS

| 名詞   | 速記        | 容易混淆點      |
| ---- | --------- | ---------- |
| MITM | 夾在中間偷聽改資料 | 目標是攔截/竄改通訊 |
| DDoS | 大量流量塞爆服務  | 目標是讓服務不能用  |

**速記：**
MITM 是偷聽插話；DDoS 是塞爆癱瘓

---

## 10. APT / 一般惡意攻擊

| 名詞   | 速記          | 容易混淆點   |
| ---- | ----------- | ------- |
| APT  | 針對性、持續性、高隱匿 | 不是隨便亂打  |
| 一般攻擊 | 廣泛、短期、機會型   | 不一定長期潛伏 |

**速記：**
APT = 有目標、有計畫、待很久

---

## 11. Zero-day / 已知漏洞

| 名詞       | 速記            | 容易混淆點      |
| -------- | ------------- | ---------- |
| Zero-day | 廠商還沒正式修補就被打   | 重點在「修補前」   |
| 已知漏洞     | 已公開且通常已有資訊或修補 | 重點在「已知可管理」 |

**速記：**
Zero-day = 還來不及補就先被打

---

## 12. SOC / SIEM / EDR

| 名詞   | 速記           | 容易混淆點   |
| ---- | ------------ | ------- |
| SOC  | 人員與流程的監控中心   | 是整體運作單位 |
| SIEM | 集中蒐集與分析日誌的平台 | 偏平台/系統  |
| EDR  | 端點偵測與回應      | 偏主機端監控  |

**速記：**
SOC 是中心；SIEM 看全域 log；EDR 盯端點

---

## 13. Event Viewer / Syslog

| 名詞           | 速記               | 容易混淆點             |
| ------------ | ---------------- | ----------------- |
| Event Viewer | Windows 內建事件檢視工具 | 看 Windows 登入、系統事件 |
| Syslog       | 通用日誌標準/機制        | 常見於 Linux、網通設備    |

**速記：**
Windows 看 Event Viewer；Linux/設備常看 Syslog

---

## 14. Facility / Severity / Priority（Syslog）

| 名詞       | 速記     | 容易混淆點                      |
| -------- | ------ | -------------------------- |
| Facility | 事件來源類別 | 例如 mail、user-level、daemon  |
| Severity | 嚴重程度   | 例如 Emergency、Warning、Debug |
| Priority | 綜合值    | 由 Facility 和 Severity 算出   |

**速記：**
Facility 看誰發的；Severity 看多嚴重；Priority 是總分

---

## 15. Debug / Warning / Alert / Emergency

| 等級        | 速記  | 記法        |
| --------- | --- | --------- |
| Debug     | 最詳細 | 除錯用       |
| Warning   | 警告  | 有異常但未到最嚴重 |
| Alert     | 很嚴重 | 需立即處理     |
| Emergency | 最高級 | 系統不可用等級   |

**速記：**
越往 Emergency 越嚴重；Debug 是最細不是最嚴重

---

## 16. Defense in Depth / 最小權限

| 名詞                   | 速記     | 容易混淆點          |
| -------------------- | ------ | -------------- |
| Defense in Depth     | 多層防護   | 不是只裝一套防毒或一台防火牆 |
| 最小權限 Least Privilege | 只給必要權限 | 是帳號/操作面的控制原則   |

**速記：**
縱深防禦 = 多層；最小權限 = 少給

---

## 17. Identify / Protect / Detect / Respond / Recover

| 功能       | 速記      | 常見例子        |
| -------- | ------- | ----------- |
| Identify | 先知道有什麼  | 資產盤點、風險識別   |
| Protect  | 先保護起來   | 存取控制、加密、MFA |
| Detect   | 發現異常    | 日誌、監控、告警    |
| Respond  | 事件來了先處理 | 隔離、調查、抑制    |
| Recover  | 恢復營運    | 還原、復原程序     |

**速記：**
先盤點，再保護，再偵測，出事就應變，最後復原

---

## 18. OWASP Top 10 / AVAS

| 名詞           | 速記            | 容易混淆點     |
| ------------ | ------------- | --------- |
| OWASP Top 10 | 常見 Web 風險類型清單 | 不是完整驗收標準  |
| AVAS         | 自動化漏洞評估系統     | 可作為驗收基礎之一 |

**速記：**
Top 10 是風險分類，不等於完整驗收清單

---

## 19. SQL Injection / XSS / CSRF

| 名詞            | 速記            | 攻擊對象        |
| ------------- | ------------- | ----------- |
| SQL Injection | 打資料庫查詢        | 後端資料庫       |
| XSS           | 注入惡意腳本給別人瀏覽器跑 | 使用者瀏覽器      |
| CSRF          | 借使用者已登入身分送請求  | 使用者 session |

**速記：**
SQLi 打資料庫；XSS 打瀏覽器；CSRF 借刀殺人

---

## 20. Webshell / 惡意程式 / 後門

| 名詞           | 速記         | 容易混淆點           |
| ------------ | ---------- | --------------- |
| Webshell     | 植在網站上的後門腳本 | 常透過上傳漏洞或 RCE 進去 |
| 惡意程式 Malware | 廣義惡意軟體     | 包含木馬、勒索、蠕蟲等     |
| 後門 Backdoor  | 讓攻擊者持續存取   | 不一定是 Web 形式     |

**速記：**
Webshell 是後門的一種，專門長在 Web 主機上

---

## 21. HSTS / HTTPS / SSLStrip

| 名詞       | 速記              | 容易混淆點             |
| -------- | --------------- | ----------------- |
| HTTPS    | 加密傳輸協定          | 有 HTTPS 不代表一定強制使用 |
| HSTS     | 強制瀏覽器只走 HTTPS   | 防降級攻擊             |
| SSLStrip | 把 HTTPS 降成 HTTP | 常見於缺 HSTS 情境      |

**速記：**
HTTPS 是有加密；HSTS 是強制不能降級

---

## 22. RDP / SMB / LDAP / DNS Zone Transfer

| 名詞                | 速記           | 常見用途       |
| ----------------- | ------------ | ---------- |
| RDP               | 遠端桌面         | 3389       |
| SMB               | 檔案分享         | 445        |
| LDAP              | 目錄查詢         | AD 枚舉常見    |
| DNS Zone Transfer | DNS 區域資料整包傳送 | 可被拿來蒐集網域資訊 |

**速記：**
3389 看桌面；445 搬檔案；LDAP 查 AD；Zone Transfer 挖 DNS 資訊

---

## 23. ntds.dit / SAM

| 名詞       | 速記      | 容易混淆點     |
| -------- | ------- | --------- |
| ntds.dit | AD 資料庫  | 網域環境的重要目標 |
| SAM      | 本機帳號資料庫 | 單機帳號資訊    |

**速記：**
SAM 是單機；ntds.dit 是網域

---

## 24. IT / OT / ICS

| 名詞  | 速記     | 容易混淆點      |
| --- | ------ | ---------- |
| IT  | 一般資訊系統 | 常重機密性      |
| OT  | 營運技術環境 | 常重可用性與完整性  |
| ICS | 工業控制系統 | OT 的核心場景之一 |

**速記：**
IT 管資訊；OT 管現場設備；ICS 是工控系統

---

## 25. CIA 在 IT 與 OT 的差異

| 場景       | 常見優先順序              |
| -------- | ------------------- |
| IT       | 常見為 機密性 → 完整性 → 可用性 |
| OT / ICS | 常見為 可用性 → 完整性 → 機密性 |

**速記：**
IT 怕洩密；OT 怕停機

---

## 26. Virtual Patching / 正式修補

| 名詞               | 速記        | 容易混淆點       |
| ---------------- | --------- | ----------- |
| Virtual Patching | 先用外部控制擋風險 | 適合無法立刻更新的系統 |
| 正式修補 Patch       | 直接修掉漏洞    | 最根本方式       |

**速記：**
Patch 是治本；Virtual Patch 是先擋住

---

## 27. 備份 3-2-1 / 單純多存幾份

| 名詞     | 速記              | 容易混淆點         |
| ------ | --------------- | ------------- |
| 3-2-1  | 3 份、2 種媒體、1 份異地 | 不是只放不同磁碟槽     |
| 單純多存幾份 | 只是數量多           | 不一定有異地與異質媒體保護 |

**速記：**
3-2-1 重點在「異地」和「不同媒體」

---

## 28. 對稱式加密 / 非對稱式加密 / 數位簽章

| 名詞     | 速記       | 容易混淆點      |
| ------ | -------- | ---------- |
| 對稱式加密  | 同一把金鑰加解密 | 適合大量資料     |
| 非對稱式加密 | 公私鑰配對    | 常用於金鑰交換    |
| 數位簽章   | 驗證來源與完整性 | 不是拿來加密大量資料 |

**速記：**
大量資料用對稱；交換金鑰用非對稱；驗證真偽用簽章

---

## 29. STIX / TAXII

| 名詞    | 速記       | 容易混淆點    |
| ----- | -------- | -------- |
| STIX  | 威脅情資內容格式 | 是資料格式/語言 |
| TAXII | 情資交換傳輸機制 | 是傳送方式/協定 |

**速記：**
STIX 管內容；TAXII 管怎麼送

---

## 30. Red Team / Blue Team / Purple Team

| 名詞          | 速記    | 容易混淆點    |
| ----------- | ----- | -------- |
| Red Team    | 模擬攻擊者 | 負責打      |
| Blue Team   | 防守與應變 | 負責守      |
| Purple Team | 讓紅藍協作 | 不是只有書面演練 |

**速記：**
紅打、藍守、紫協調

# 超短版口訣

## 工具類

* **CVE = 編號**
* **CVSS = 分數**
* **SAST = 看碼**
* **DAST = 打系統**
* **SIEM = 看 log**
* **EDR = 看端點**

## 攻擊類

* **Brute Force = 一帳號多密碼**
* **Password Spray = 一密碼多帳號**
* **SQLi = 打資料庫**
* **XSS = 打瀏覽器**
* **CSRF = 借登入者送請求**
* **MITM = 攔截**
* **DDoS = 塞爆**

## 管理類

* **SSO = 方便**
* **MFA = 強驗證**
* **VPN = 安全通道**
* **Defense in Depth = 多層**
* **Least Privilege = 少給**

## 日誌類

* **Event Viewer = Windows**
* **Syslog = Linux/網通**
* **Debug = 最細**
* **4625 = 登入失敗**
